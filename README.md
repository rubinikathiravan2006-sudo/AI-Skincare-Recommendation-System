# AI-Based Skincare Recommendation System
This project recommends skincare products and routines using Artificial Intelligence and Machine Learning techniques based on user skin type and skin concerns.
## Technologies Used
- Python
- Machine Learning
- Pandas
- Scikit-learn
## Features
- Personalized skincare recommendations
- Skin type analysis
- AI-based product suggestions
## Objective
To help users identify suitable skincare routines using AI.
